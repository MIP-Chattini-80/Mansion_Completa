README DEL PROJECTO "MANSION DE LOS ECOS"

Este readme es para mostrar los pasos que hemos hecho para crear nuestro poyecto del juego "Mansion de los Ecos", el cual nos ha dado varios dolores de cabeza, pero que al menos lo avanzamos
aun buen puerto, si es que lo logramos, ehe. (Ehe te nandayo!!)

Bitácora de rojas: estoy sufriendo, pero logramos avanzar.

V1 - El primer Commit subido fue de la creación de los 3 microservicios principales habitaciones (Mansion-Electric-boogaloo), personajes y jugador-progreso, previo al siguiente paso.

V2 - La creación del swagger dentro de los microservicios se creó correctamente.

V3 - Creación de las ramas "Eureka" y "Gateway"

V4 - Revisamos los Eureka y Gateway y los modificamos al respecto de nuestro proyecto, para que no haya errores

V5 - Las ramas feature están desordenadas, pero cada una muestra un tipo de avance durante el proyecto, para compara lo que mi compañero subia, ya que todavia nos cuesta adaptarnos al
github, y varias cosas del proceso se perdieron o no se subieron por lo mismo, y que a veces fallaba la subida.

V6 - La creación de un archivo .bat que nos ayudará a activar todo el programa, y funciona bastante bien, ya que cada servicio funciona bien.

v7 - se agregaron nuevas cosas del lore y dialogos, para que la historia fluya y se agregaron más cosas para poder guardar nuestra partida desde donde quedó.

v8 - se modificaron algunos aprametros del gateway para que se pudieran ver los datos en el navegador.
